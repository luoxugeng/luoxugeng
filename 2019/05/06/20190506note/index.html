不适应感可以变成是创造的动力.

发表于 2019-05-06 更新于 2021-03-19 分类于 饭否 Disqus：
本文字数： 658 阅读时长 ≈ 1 分钟
这里我指的是美好事物从你身边被拿走,使得你只能用次品的那种不适应感.

之前写过关于翻墙的文章安全技能树简版:好奇心– 翻墙的历史, 今天看到编程随想如何让[不支持]代理的网络软件,通过代理进行联网(不同平台的 N 种方法) 里面的方法不得不说让我打开眼界,虽然术语我都是知道的,但从来没想到过可以这样组合使用.技术修养高低立现.

关于翻墙,我这样做其实一开始只是出于一种 不适应感.

我上大学那会2011年,那时候,Google虽然退出中国,但搜索还是能用.在2013年左右,Google搜索就彻底不能用了, 而我有一段时间只能使用百度.搜索质量之间的差别引起了我的强烈不适应.当时只有一个念头:我怎么才能用Google.

天朝所害怕的,想要掩盖的历史,想要封禁的假想敌,说实话,我这样一个老百姓,关心么? 但反而在这个过程中知道了很多人的贡献已经被封口(详情看 翻墙的历史), 这些活在当下的人才是我关心,敬佩的

墙,不是必要的.恐惧并不会因为你筑起一堵墙而消失.

经验过的不适应感还有一个: 从互联网公司去到传统的投资银行.因为互联网公司有多年的技术积累,很多事情存在是让你提高工作效率的(当然也有繁琐的流程),这些平时习以为常的基础设施只有当你不再体验的时候你才会体会他们的价值.这时候的不适应感,或许是你能改造这家公司的资本.这也是为什么很多小企业会青睐大公司出来的原因.

** 因为你知道哪些是美好的事物,你体验过,你知道事情不仅仅可以这样,这种不适应感会变成你改造的动力.**

另一个创造的动力我认为是对工具的追求.你作为一个创造者, 知道哪些事情让机器来做是更合适的,这也会使得你拥有创造力.当然这是另一个话题.
