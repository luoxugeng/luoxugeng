在吃饭的时候观察自己

发表于 2019-06-11 分类于 每日写一点 阅读次数： Disqus：
本文字数： 190 阅读时长 ≈ 1 分钟
今天一边吃饭,一边发呆.感觉灵魂出窍了.

今天一边吃饭,一边发呆.就好像你有一个意识脱离了你本身,它就在你旁边,看着你吃饭,怎么夹菜,怎么咀嚼.就静静看着. 你本身好像一台被下了指令的机器人,真正的你正在看着你怎么执行.

然后它会看到你在看着别人,没有任何判断,也是在看着别人走过,吃饭.

可能是最近机器人系列的小说看多了.

有意思的是, 在工作上这种状态也可以复现( 因为就是机械重复, 但当我意识到这一点的时候,机械就被自己打断了 )

把问题描述清楚,而不是解决方案
不要约束人的行为,要让机器发现不合理
