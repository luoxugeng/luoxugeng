    回顾

  • 熟悉Mac. 关键是我赋予了你一个观念: Mac 是效率生产器,不要将就,一定可以有更好的解决方法的.

  • 介绍一些生产软件,例如: 获取信息的渠道,例如inoreader. 存储信息的方法:Evernote

  • 知道为什么要科学上网; 能使用 Google,能使用 Google 高级搜索

  • 在 iTerm2 使用常用的 bash 命令; 配置你的iTerm2 ,让自己用的舒服.

  • iTerm2 中重点介绍了 git, 作家或者程序员都会喜欢上的.

  • 编辑器 Vim.

  • 最后要是你足够勤快, 你应该已经完成了个人博客静态博客. 输入 XXX.github.io 应该能访问到你个人博客.

当然,要是实现了个人域名,那就更棒.

接下来

知道创宇的技能树 基础部分还有以下几个话题没有提到

  • 翻墙

  • 云服务

  • 隔离

  • 暗网

  • web 安全

我觉得麻烦的点在于,这几个话题技能是相互夹杂,要是我一个不小心把握不好,容易把读者绕晕.下面提出我的学习方案,重要的是为什么,为啥而学.

首先: 翻墙 和 云服务

翻墙的推荐方式: SSR 只是一个结果.翻墙整个过程就像一个抗争史一样,我很乐意传播我搜集过来的资料,让读者知道有这么一回事.

然后我还计划顺便把我的计算机网络总结一遍.可能开一个番外篇啥的.因为只有你对整个网络原理有一个不错的了解,你才能走的更远.用工具的人到最后都有自己的冲动制作自己的工具.

当你拥有了一台 vps 的时候,其实你是有能力把它变成自己的私有云. 云服务其实没那么复杂,当然一个人用和成千上万人用不是一个维度.

接下来是隔离

这里指的是网络隔离.我将会用 virtualBox 来介绍.当然可以闲扯一些其他隔离方式.

为什么要隔离? 或者说隔离可以怎么用?

  • 我假设你是一个十恶不赦的混蛋,例如偷吃了隔壁小朋友的糖葫芦之类的. 你这个恶人想隐蔽上网避免被追踪;

  • 或者你是一个好奇心旺盛闲到蛋疼没事干,就想看看暗网有啥;

  • 作为一个码农,在虚拟机搭建一个干净的环境开发不是很自然的事情么? 最后一个才是王道,老老实实干活多好….

暗网

暗网其实没有你想象的神秘.我通过搜索都可以找到门道.难的地方在于,你真想做到完全匿名,那就要多花点心思了.上面可以当做是在为这个铺垫.(搞安全的是不是都这么多见不得光的东西呀,好奇…)

当上面讲完,整个基础必备告一段落.然后我们只完成了整个思维导图的20% 囧囧囧…
