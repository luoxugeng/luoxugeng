博客纪事

博客一些关键节点,可以记录一下.博客是2017年4月成立的,在这之前的文章都是从别的地方迁移过来的.强调一下

博客纪事

创立之前计划博客公众号两边同步,但刚好看到这篇博文:为什么我不在微信公众号上写文章, 于是作罢.
写作,只是我的业余爱好,一个倾诉观点的地方.一个让我唠叨无病呻吟的地方.哈哈….

详细教程善用搜索引擎, 可以入门 Google Hack,不管是不是程序员,这个技巧是很有用的. 清楚自己需要干嘛, 接下来总有办法解决.

  • 2016-07: 里程碑! 博客成雏形
    博客托管 在个人 Github ;
    博客框架 使用Hexo Hexo 是由Node.js驱动的一款快速、简单且功能强大的博客框架;
    博客主题 是jacman;
    个人域名 通过GoDaddy购买;
    DNS解析 使用dnspod;
    dispute 作为自己的评论系统;
    写作 由 Markdown 强力驱动,比 Latex 容易入门多了.入门链接点这, 写作编辑器 一开始 Mou, 后停止更新, 现在使用酷酷的 Atom,一个号称比 Vim 还 Emacs, 比 Emacs 还 Vim 的新一代编辑器.

  • 2016-08

提交 sitemap 到 Google 和 baidu ; 让自己的网站可被搜索. 总有一天, 搜索 nemo, 排行榜第一会是我的网站!

使用 站长统计 统计网站当前人数

  • 2016-10

~ 选择七牛云 作为自己的图床,使用官方API上传本地的图片,避免每次手工上传.

  • 2017-02

增加 打赏 功能, 感谢Qiu Hu的教程,我打赏6.66~~

使用 不蒜子 统计网站和文章的浏览次数.(恩,历史数据就这样没有了…)

使用MathJax插件解决数学公式显示问题.安装参考博客 ; 符号帮助手册

  • ** 2017-10**

购买了新的域名, 在机器上搭建了迷你版的微博. 我只是自恋到想有一个地方唠叨

DNS解析改成cloudflare,同时让自己的网站支持 https

  • 2017-12

把自己以前在QQ空间的日志搬到这边来,我们沟通的工具一直在变

关闭网站的统计功能.也不想花心思在人群上,因为可以直接评论,也可以直接找我.

修改自己的打赏二维码,可以点击进去看看,你会发现有意思在哪里.

  • 2018-06

使用hexo-douban 增加我的豆瓣影片和书籍,非常好的一个功能,赞赏作者23.33

自从搭建了自己的 git 服务器, 大部分文件在同步同时拥有了自身的备份,不管是在家里还是工作的地方,思路可以不中断继续下去.这种 workflow 太赞了.
