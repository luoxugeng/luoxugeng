科学上网

十九大期间,服务器基本连不上去,更别说科学上网了.后来恶补了 SSR 的基础知识(关键词:socket5 ,流量识别).

解决的思路:

  • 服务商网段没有被封,换一台机器.
  • 服务商被封了, 换一个服务商.例如有一个全球最大电商网站,老大是个光头的那谁,很多中小型企业绑在上面,封的话殃及太大.第一次注册还可以免费一年
  • 换一个思路,既然SSR 是基于socket5, socket5的流量特征太明显,这条路走不通,那就往 网络更底层的地方走: vpn.有意思的是,在这个过程接触了 tor.

整个过程我好像卷入了什么斗争一样.诡异的很.还了解到 SSR 作者,他开源这个工具一开始的初衷,对开源的看法,后来有人拿他软件商用的他内心的失望, 最后因为被请去喝茶放弃项目的声明.这都是3年前的事情了.人不在,代码却还在发挥作用.程序员伟大的地方在这里.恩,我们今天的便利其实是建立在很多无名英雄之上的.他们做了很多事情,我们又做了什么呢?

搭建了自己的 git 服务器.

很多东西我想保留更改记录, 国内第三方的我信不过. github 还好,但个人私有仓库只有一个.自己动手, 丰衣足食吧.现在我就把我的博客文章,代码,系统的一些配置文件放在上面.放在上面.看过有一个哥们在 github 上托管自己的博客源文件的,自己配置文件的 key 都传上去了,智商堪忧.发邮件跟他说了.

类似微博

我是一个话痨,在 QQ 空间话痨–>在微博上话痨–>在微信上话痨.最近已经不知道在哪里话痨了, 有时候就是希望有一个地方记录自己的想法,希望别人看到,但又不希望被所有人看到,或者不希望现在就别人看到.所以有了下面这个东西的存在.在 VPS 上安装WordPress 可以搜索一下 LNMP,把 WordPress 主题换成类似微博的就搞定了.手机上装一个 WordPress 客户端,以后随时随地想写就写了.而且,只属于你.
