天使或者恶魔:马基雅维利测试

发表于 2017-03-28 更新于 2019-02-14 分类于 StoryTeller Disqus：
本文字数： 767 阅读时长 ≈ 1 分钟
马基雅维利衡量的是一个人能在多大程度上不受个人情感影响,跳开传统的道德思考标准,以及驱使他人为其工作的能力.我觉得这种性格是有两面性的

马基雅维利人格会根据一个人的际遇强化或者减弱的.如果你的性格比较弱,你可以时不时把下面的语句拿出来念几遍,让它刻印在自己脑海里.这是对自己性格潜移默化的一种方式,有可能走火入魔,可能找到自己的 another side.

马基雅维利认同的

  1. 永远不要告知别人你行为的真实原因,除非这样做对你有利
  2. 驱使他人最有效的方式就是说他们愿意听的话
  3. 绝不让善恶的标准来束缚自己的行动
  4. 不相信人的本质是善良随和的
  5. 大多数人一有机会,都会为非作歹
  6. 只会选择性诚实
  7. 如有必要,说谎就是一件很自然的事
  8. 大体上,如果不被逼迫,没人会真正努力工作
  9. 谦虚诚实,骄傲虚伪,都不过是达到目的的一种手段
  10. 当需要别人为你做事的时候,更在意是否有效,而不在意是开诚布公,还是虚与委蛇.
  11. 越是高层的人,越没有道德可言.
  12. 完全相信一个人就是自讨苦吃
  13. 罪犯和平常人最大区别,就是罪犯智商不足,被抓了
  14. 大多数人是懦弱的
  15. 知道该逢迎哪些有权有势的人
  16. 人无完人
  17. 每一分钟都有人上当受骗.
  18. 要想领先,可以不择手段
  19. 人有权力决定自己的生死
  20. 比起失去至亲,大多数人对财产的损失记忆更持久
