数字人生系列(一)

下面是每一集快速预览

数字人生(一):成为百万富翁的潜质

  1. EQ
  2. 谈判能力
  3. 创造性思考
  4. 适应能力
  5. 坚持
  6. 小睡一会
  7. 刻意延迟

数字人生(二):大说谎家

大多数人都说过谎,尤其是对陌生人.
我们其实是习惯于这个充满谎言的世界.
我们能做的就是识别出不好的谎言,别介意善意的谎言.
最后,说谎伤身.(头痛. 经常说谎的人会比诚实的人多56%的概率有生理问题,尤其是头痛)

数字人生(三):当大老板特质

  • 创造力和乐观
  • 组织完美团队
  • 用个人魅力领导

数字人生(四):风险管理的方法

  1. 了解自己有多大胆; 追随人群;鼓励自己内心的冒险者
  2. 找出好坏风险之间的细微差别. 避免盲目冒险,了解赔率,跟随热情
  3. 细心思考平常生活

数字人生(五): 混蛋与好人的抉择

  1. 当一个没心没肺的混蛋你可以:赚更多钱;体魄更强健;压力更小;更容易获得提拔
  2. 想象拥有权力和财务,是当一个混蛋的关键因素.
  3. 当一个好人你可以: 提高团队协作效率;更长寿;增加成功约会概率.
  4. 阅读思考反思,同理心可以帮你 be nice.

数字人生(六): 求生技巧

  1. 选择合适的避难场所
  2. 寻找干净的水源
  3. 利用日常物品生活
  4. 学会团队协作

数字人生(七): 解码能力

解码能力的三个级别:

  1. 专注,在嘈杂的信息中找到特定信息

  2. 多任务存储:

  3. 模式识别: 利用已有的知识解决问题

    生活小建议:

    1. 信息尽量水平展示,因为我们眼睛习惯左右扫视
    2. 2g/ 天鱼油可以提高23%的工作记忆能力
    3. 1.3杯/周 的高级香槟可以增强你的空间工作记忆
    4. 大脑卡住的时候,休息个30min.

数字人生(八): 迷信行不行

发现模式和试图控制环境的天生欲望使我们发展出迷信

如何利用迷信

  1. 仪式感:让生日蛋糕更好吃,唱生日快乐歌和吹蜡烛
  2. 幸运物:类似安慰剂效应
  3. 幸运儿心态: 自认为幸运的人做事效率会被自认为倒霉的人会好

数字人生(九): 情绪控制

  1. 没有充足的休息,人很容易抓狂,而且最好的方式是自然醒
  2. 减少你上下班所花的时间
  3. 工作时间同事的干扰你应该: 以坚定但不挑衅的方式表达不满,借以释放怒气
  4. 工作中一次只处理一件事
  5. 下班回家后,轻吻你的伴侣,拥抱某人,单身的可以唱歌.
  6. 睡觉前远离蓝光(手机等电子产品)

控制情绪的小技巧:

  • 尝试深呼吸一口气,从一数到十
  • 重复念一个词或者短语,借此转移注意力
  • 放一张你喜欢人的相片在旁边
  • 薄荷可以刺激脑部, 降低驾驶期间的焦虑和疲劳,让你感觉更好.
  • 做运动, 活动身体是释放怒气的好方法
  • 花时间学习正念冥想
  • 尝试养宠物,但不建议跟宠物睡在一起.

数字人生(十): 体魄训练

强悍体魄三个衡量要点:

  1. 疼痛忍耐度

    • 年纪越大,忍耐疼痛的能力反而会更强
    • 尝试洗冷水澡,疼痛的时候爆粗或者听音乐
    • 获得能量: 2:1 碳水化合物和蛋白质,例如巧克力牛奶

  2. 体力
    让自己看起来很强悍,可以让你真的变强悍.

    • 让自己看起来能赢: 采取看起来有力量的姿势,例如把手举起来两分钟
    • 像生物争夺一样,让自己看起来身形巨大
    • 每天练习这些看起来让你变强大的姿势,实际上真的会使你变自信强大

  3. 心理坚韧度
    通过直面你的恐惧来增强你的心理坚韧度.
    最好的方法是结队一起克服恐惧:看到有相同恐惧的人克服面对,可以增加其他人克服的勇气

数字人生(十一): 如何活得更久

建议:

  • 刷牙(牙龈炎会提高心脏疾病的风险)
  • 防晒,戒烟 (80%的皮肤老化来自于日晒, 吸烟不仅会加速老化过程,同时减少寿命)
  • 骑单车上班
  • 尝试站立办公
  • 适量饮酒有益身心,每天1~2杯咖啡有益身心
  • 提升性质量
  • 常笑:压力更少

数字人生(十二): 提高吸引力

产生吸引力的因素:

  • 外貌上:包括天生的脸蛋身材和后天的服饰搭配
  • 个性上: 幽默感 和 模仿性
  • 价值观相近

数字人生(十三):三个关键,打开你的英雄霸气

关键1:同理心强
关键2:微调你的视觉反应时间, 建立情境警觉度
关键3: 出任务的时候: 呼吸,放松(可以通过大叫来释放压力)

数字人生(十四): 提升说服力的技巧

欺骗和谎言:短期之内达到你的目的,但长期却对你不利.
影响说服力的因素可以从三部分入手:

  1. 说什么:稀缺性,体验性,潜力
  2. 谁说的:权威和同侪压力
  3. 如何说: 肢体,视觉

数字人生(十五): 谎言识别

三个关键

  1. 可以相信别人,但也要学会辨别
  2. 注意你的接触面
  3. 运用你的知识

初次约会假装坠入情网,可以增加再次约会的机会两倍

建议:
学习提升你的视知觉: 例如打14小时的电动游戏,可以提升你30%的视知觉能力
尝试玩整人游戏
学习读心术: 读心术的本质是观察能力和统计学.
