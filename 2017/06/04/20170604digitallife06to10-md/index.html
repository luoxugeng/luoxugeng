数字人生(六): 求生技巧

  1. 选择合适的避难场所

  2. 寻找干净的水源

  3. 利用日常物品生活

  4. 学会团队协作

    • 在野外,遮蔽所比食物和水更重要
      没有食物,人可以撑46 周,没有水,可以活35天.但体温下降会在短时间内夺走你的生命
    • 有超过1400中可食用的昆虫, 其中有344种是甲虫.但要避开毛茸茸或者颜色靓丽的品种,可能有毒.
    • 干净水源( 保险套 一加仑的水只要加上加上1/4的漂白水等待30min有净化的作用)
    • 河水湖水至少要煮1min才可以饮用
    • 学习利用日常物品生火(可乐罐 磨光 太阳 反射)和灭火(苏打水,醋, 塑料瓶,纸巾 )
    • 在野外,下雨天外出的风险:雷电.从看到闪电到听到雷神,5s以内说明电源很近,不适合外出.
    • 学会辨别方向:缺乏路标的情况下,不要轻举妄动.因为我们通常是在一个范围内不断绕圈
    • 断电之后,冰箱里面的食物大概可以保存4h
    • 肉类,蛋类在温度超过40度下2小时,就应该丢弃掉
    • 空难发生, 靠近机尾的位置可以使你有超过40%的几率死里逃生
    • 团队协作: 能力类似的两个人做决策比一个人有93%的概率更优秀; 社会关系较强的人, 生存率可以增加50%

数字人生(七): 解码能力

解码能力的三个级别:

  1. 专注,在嘈杂的信息中找到特定信息
    80% 的车祸源于注意力分散; 73%接受考量的奥运冠军和教练指出专注力是提升表现的关键
  2. 多任务存储
    不建议多任务并行处理,但要学会多任务存储.记忆细节,接下来才能分析处理
  3. 模式识别: 利用已有的知识解决问题

生活小建议:

  1. 信息尽量水平展示,因为我们眼睛习惯左右扫视
  2. 2g/ 天鱼油可以提高23%的工作记忆能力
  3. 1.3杯/周 的高级香槟可以增强你的空间工作记忆
  4. 大脑卡住的时候,休息个30min.
  5. 屏住呼吸思考是有一定道理的,男性可以压住左鼻孔,女性则是右鼻孔, 这在空间识别上男性可以提高8%的准确率,女性则是16%
  6. 10% 玩解谜游戏的人在过去一年会获得晋升. 28% 的人获得加薪

数字人生(八): 利用迷信

发现模式和试图控制环境的天生欲望使我们发展出迷信

如何利用迷信

  1. 仪式感:让生日蛋糕更好吃,唱生日快乐歌和吹蜡烛

  2. 幸运物:类似安慰剂效应

  3. 幸运儿心态: 自认为幸运的人做事效率会被自认为倒霉的人会好

    • 带有13数字的房子平均会便宜6k刀
    • 13号周五,美国人会倾向于减少消费,呆在家里减少损失.
    • 自认为幸运的人做事成功率会真的变高. 这跟焦虑感有关, 认为自己不幸的人无意中降低了自己处理突发意外事情的能力.
    • 日蚀会让股市下降,因为美国人将它和灾难联系在一起.

    了解你玩的游戏

    中了七次乐透的人说了他的三个原则

  • 原则一:不要靠运气.一定要了解你玩的游戏.
    他跟我们讲了选数字的三个策略:电脑随机选,选自己幸运数字和生日,自己随机选.
    多数人用电脑选,所以每次中奖机率都接近0
    用幸运数字的方法问题在于,数字范围本身有局限性.而且,很多人这样做,中奖后你会跟很多人瓜分

正确做法是,自己挑选数字,并坚持固定投这些数字

  • 原则二:定预算.只花你多余的钱

数字人生(九): 情绪控制

  1. 没有充足的休息,人很容易抓狂,而且最好的方式是自然醒
    人在睡眠中醒来的前几个小时,,称之为:快速动眼睡眠. REM. 在大脑准备醒来就被闹钟吵醒的,会缩短 REM, 损害注意力, 警觉力, 集中能力,思考力和解决问题的能力
  2. 减少你上下班所花的时间.
  3. 工作时间同事的干扰你应该: 以坚定但不挑衅的方式表达不满,借以释放怒气
    例如有人在你的区域大声说话(吸口气,然后坚定地请他们离开:表示理解,你的问题,你的要求) 要是对方是你的老板,立场坚定,但建议语气委婉一些. 要是老板的评论是针对你个人, 问清楚他的需求; 要是是对你的批判,了解他对你的期望.
  4. 工作中一次只处理一件事
    任务串行很让你更有生产力,同时增加你工作的满意度.
  5. 下班回家后,轻吻你的伴侣,拥抱某人,单身的可以唱歌.
  6. 睡觉前远离蓝光(手机等电子产品)

控制情绪的小技巧:

  • 尝试深呼吸一口气,从一数到十
  • 重复念一个词或者短语,借此转移注意力
  • 放一张你喜欢人的相片在旁边
  • 薄荷可以刺激脑部, 降低驾驶期间的焦虑和疲劳,让你感觉更好.
  • 做运动, 活动身体是释放怒气的好方法
  • 花时间学习正念冥想
    在心中描绘出你觉得能放松的地方或者情景,静坐冥想
    找一句咒语,反复地念
    把你的注意力放在呼吸上
  • 尝试养宠物,但不建议跟宠物睡在一起.

拒绝养成方式: 街搭

这个很有意思

面对拒绝时关键是要:保持镇静 .
特意让自己在大街上被拒绝,视频教我们接受拒绝的三个要点:

  • 控制自己的反应: 被拒绝只是一个信息,它取决于我们本身对它的解读.
    很多人以为自己没有选择(被拒绝), 但你可以选择当前这个拒绝对你的含义(完全的失败或者是一次不错的尝试)
  • 不要认为拒绝是针对你个人的.
    了解被拒绝最好的方法,就是亲身体验. 再多的理论也不会改变你的人生,除非你亲身体验. 你必须亲自体验被拒绝. 面对你的恐惧, 走出你的安逸圈.
  • 永远积极行动. 让对方看到你的认真.
    恩,不要看到妹就撩.每一次你都要问自己,我是认真地想要认识眼前这个人么?

数字人生(十): 体魄训练

强悍体魄三个衡量要点:

  1. 疼痛忍耐度

    • 年纪越大,忍耐疼痛的能力反而会更强
    • 尝试洗冷水澡,疼痛的时候爆粗或者听音乐
    • 获得能量: 2:1 碳水化合物和蛋白质,例如巧克力牛奶

  2. 体力
    让自己看起来很强悍,可以让你真的变强悍.

    • 自己看起来能赢: 采取看起来有力量的姿势,例如把手举起来两分钟
    • 像生物争夺一样,让自己看起来身形巨大
    • 每天练习这些看起来让你变强大的姿势,实际上真的会使你变自信强大

  3. 心理坚韧度
    通过直面你的恐惧来增强你的心理坚韧度.
    最好的方法是结队一起克服恐惧: 看到有相同恐惧的人克服面对,可以增加其他人克服的勇气
