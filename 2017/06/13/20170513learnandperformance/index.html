TED:如何表现得更好

发表于 2017-06-13 更新于 2021-04-29 分类于 书影空间
本文字数： 1k 阅读时长 ≈ 1 分钟
有意识地花更多的时间在Learning Zone上,这是我们成长的关键; 在需要高效专注的时候,应该处于Performance Zone, 并在之后反思整个过程,力求精益求精,是我们成长的另一途径
  • Learning Zone VS Performance Zone

  • 建议

主题跟刻意练习相似. Talker的内容主要有:

  1. 能力区域可以分为两部分的:Learning Zone(学习区域)和 Performance Zone(执行区域).
  2. 高效的人和团队都会刻意地让自己在这两个区域交替进行.
  3. Learning Zone 关系到我们的成长,并在未来影响我们在Performance Zone的表现
  4. Performance Zone 能够让我们以高效的方式聚焦于当前的工作,这时我们需要反思.

Learning Zone VS Performance Zone

Learning ZonePerformance Zone
ImprovementExecution
Haven’t mastered yetHave mastered
Mistakes to be ExpectedDoing thing as best as we can

两个例子:

Demosthenes: 有意识地去学习
古希腊的律师和演说家.他有意识地根据不同的场景锻炼自己缺失的能力.

  • 前提: 他已经掌握了大量法律知识和演讲技巧
  • 发现自己演讲会不自觉抖肩膀: 拿东西压住
  • 说话结巴: 嘴巴里放几块碎石
  • 需要在嘈杂环境辩论: 到海边大声说话

Beyonce: 在练习中精益求精.
每天晚上在舞台表演后都会回放录像带,看看哪些是可以做得更好的.这类似于我们每天的工作,即使是相同的工作,我们也可以选择不一样的态度去面对.

建议

Talker指出: 正是对周围环境的不安全感,导致我们会花大量的时间在我们的执行区域上.
职场上雇主希望我们可以有高效输出,学校希望我们可以通过标准考试测试,这些都会让我们不自觉地待在我们熟悉的执行区域,而不敢,或者根本没有那个意识想去探索新的学习区域. 这样导致的结果是:个人渐渐失去自己的竞争力,公司也失去创新能力.

有意识地让自己处在学习区域:

  • 要有坚定一个信念: 相信我们是可以继续成长的,(成长型思维模式)
  • 我们必须要有一个明确的目标,因为练习是会花费我们时间和精力的.
  • 即时反馈:要知道如何改善,做些什么去改善. 这使得我们的练习不再是大量时间的重复,而是有意识沉淀成我们自己的技能. ( 不像以前学乌克丽丽那样只会一遍又一遍地重复.我们必须知道哪里表现地怎么样,并知道怎么去改善,这才是刻意练习)
  • 我们必须处于低风险的情境下.犯错的结果不能是灾难性的. 在这个过程,我们可以咨询导师的意见,也可以向别人分享自己的学习经验. 这些都可以增强我们这次或者下一次处理类似事情的信心.
