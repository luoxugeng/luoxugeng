关于西部世界

发表于 2016-12-06 更新于 2020-04-09 分类于 书影空间 阅读次数： Disqus：
本文字数： 2.4k 阅读时长 ≈ 2 分钟
西部世界,一部可以慢慢回味的神剧.

2016:Westworld (西部世界)实在是太精彩了,尤其是最后一集. 虽然我是到最后在发现是双时间线(前面有大神分析,但到最后一刻我才能肯定),还有就是Ford 的反转(他对Arnold的情感,救赎,我只想唱:如果这都不算爱…) . 现在再回过头来看前面的一些剧情,才发现原来背后隐藏的寓意是那么深刻.

实现梦想的人不会真的快乐的

有一集是 Ford在地下室跟 Bill 聊天, 一开始我觉得有点突兀,以为是闲聊.但实际上不是.撇开双时间线来看, 你会发现主线其实没过多少天. Ford貌似是在回忆小时候的事情,但他说的是他自己.他就像那只灵缇犬,用30年的时间去追寻一个结果,去弥补了一个错,但等到真的走到尽头时,他作为人,一个活生生的人,愣了.我想是在这之后他才决定要让Dolores让他安息.因为这是Ford能想到最好的谢幕方式了.他可以活在他一生的作品里面而不是他自己本身.

你想知道我见过的最悲伤的事情吗?灵缇犬就是一种赛犬.它们终其一生都在赛场里一圈一圈地跑,追逐着那个毛毡布做的小兔子.有一天,我们带它去了公园.之前,父亲曾提醒过我们它跑得很快,千万别撒手.不过,我们还没抵挡得住诱惑.想看看它跑得到底有多快.于是,哥哥取下了狗绳.就在那个时候,灵缇犬看到了一只猫.我猜,它一定是把那只猫当成了毛毡布兔子了.它就立即撒腿狂奔过去.那只狗全速狂奔的样子,是我有生以来见过的最美妙的景象了.最后,它抓住了那只猫.令大家惊恐万分的是,它杀死了那只小猫,把猫撕得粉碎,然后它就坐在那里,一脸懵圈.那条狗一辈子都在努力要抓住那个东西.现在,(它抓到了之后),它不知道该如何是好了.

正是苦难让我们感到真实

Ford 让 Host 觉醒的手段:磨难.
Arnold 失去了他的儿子. Ford 失去的是Arnold.当Ford看到倒在地面上的Arnold尸体,我相信他是痛苦万分的.
跟自己最亲近的拍档最后一次对话竟然是一次争吵.多年以后,当他第一次唤醒Bernard,说了一句: hello,my old friend. 简单的一句是包含了多少复杂的情感?
Ford用接下来的三十年去纠正他犯下的错误.Arnold死去的时候其实并没有找到让Host觉醒的方法,但Ford 找到了.我不知道Ford是什么时候意识到自己错了,也不知道过了多久才发现让 Host觉醒的方法:让他们受尽苦难.但这些我相信这些是Arnold的死去催化的.因为苦难导致内心强烈的波动,甚至是颠覆自己以前建立的价值观.可能是工作的剧烈变动,可能是生活中某位至亲人的离去,可能是自己身体的变化. 这样的动荡,最直接的结果就是打破你日常的 Loop. 我们都不喜欢改变,我们会倾向于在熟悉中寻找安全感.但这些苦难却最直接有效的,它先让你无助,绝望.让你眼睁睁看着自己所珍视的人或事物被人从夺去. 即便是一滩死水,这时候也应该要激发出翻江倒海的能力.

短短的一个月,我经历了丢包事件和考核背锅.年会上带去的包,因为喝醉落在班车上.班车司机见财起意,一口咬定说没拿.最后我怎么拿回来的? 有一种运气的成分在里面,我不断让公司的人出面施压.因为我知道,以个人的名义,对方是完全没有动力为我找回东西的.司机一口咬定包不在,我威胁他说我已经报警了,(其实警察根本不受理)我就恐吓他,就是他拿的,我要他吃不了兜着走.当时也没把握,只是死马当活马医. (报警的时候,不要说你喝醉了,不要说一些对你不利的,即使撒谎,也要让警察把案立了再说) 考核背锅,但接下来却强迫这我去思考,对整个大环境,对自己职业生涯,什么是我要的.以我的腾讯生涯作为代价换来的.但不见得是一件坏事.这样的遭遇把我对自己人生应有的规划推到我面前,这种打击其实是越早越好! – 增加于2017年2月14号

寻找自我的旅途

William在西部世界的黑化,我是能够理解的.他有了不断拔枪的理由:他在寻找Dolores.
William在杀戮中找到了另一件东西–他自己. 但我觉得他只是找到了一部分自己,他找到的这部分自己,可以让他冷酷无情地去追逐权力财富.人一旦掌权,就会忍不住想胡作非为.在乐园他发现他可以比想象的还要冷酷,我猜测正是这种改变让他再回到现实世界后能接管Delos的一大原因.在现实世界,他用残酷的手段成为了别人口中成功的企业家,慈善家.是一个披着天使外衣的魔鬼.但在乐园,却是相反,他在乐园的残酷行为只是为了一个简单的目的: 他希望 Host 能反抗. 他想抓住的是 one true thing.

游客不可能成为输家.也就意味着这一切都是谎言…假如这些不是真的,那它对我啊来说什么都不是.

死者永生

肖邦,贝多芬, 他们已经融入在他们的作品中. 有些人,即使肉体消亡,他遗留的思想仍然在影响着活着的人.

我们其实也生活在西部世界中

游客 VS Host 对比于 观众 VS 西部世界
Host 所生活的乐园,每一个细节都有其特殊含义,不像游客们生活的世界一片混沌.乐园的一切都可以是真实的除了它存在本身.
像我这样的观众,我会一遍又一遍的去看西部世界,代入里面的场景,去思考里面每一个细节的含义.有没有想过其实西部世界的意义已经超越了它本身,他可以是我们日常生活经常依赖的东西.例如游戏,例如手机. 我们每一个观看的人,其实最后是走向了William 的道路: 在虚拟的世界里面希望找出一种真实.在这过程也希望得知造物主的想法. Maze, 到底意味着什么? 结果答案竟是一个 toy.

游客不会因为一眼就能看到的东西再来,他们会再来是因为那些细微的地方.他们会再来是因为他们发现了他们以为没人注意到的东西,一些他们深陷其中不能自拔的东西.他们不是在寻求一个告诉他们是谁的故事,他们已经知道他们是谁.他们来这里是因为他们想要一窥自己还有可能成为什么样的人.
